# 🍽️ TenantTable API

TenantTable is a **multi-tenant Restaurant Point of Sale (POS) backend** built with **Spring Boot**, designed for startup restaurants. The project focuses on providing a secure, scalable, and maintainable backend architecture that can grow into a complete restaurant management platform.

---

## About

TenantTable is being developed with a strong emphasis on **clean architecture**, **security**, and **production-ready practices** rather than rapid feature development.

The project serves as the backend foundation for a cloud-based restaurant POS system, where each restaurant operates as an independent tenant while sharing the same application infrastructure.

---

## Architecture

The project follows a layered architecture to keep business logic, persistence, and presentation concerns well separated.


Current architecture includes:

* Layered Architecture
* DTO-based API Design
* Repository Pattern
* Global Exception Handling
* JWT-based Authentication
* Asynchronous Task Processing
* RESTful API Design
